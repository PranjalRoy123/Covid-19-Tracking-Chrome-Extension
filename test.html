Coronavirus Tracking Chrome Extension

Chrome Extension

What it is...

This is a simple chrome extension which allows you to track the current worldwide coronavirus data. This works using HTML, CSS, JS and an api.

How to use...

Currently we cannot download this extension directly from the webstore in Chrome. But we can run it locally in our browser by:

  • Download or clone the repository.
  • In your Chrome browser, go to chrome://extensions
  • Click on the Load Unpacked button and choose the folder you just downloaded.
  • To use the extension just click on the jigsaw symbol beside the url bar and select the Covid-Tracker extension.
  • Enjoy!!!

API

This extension uses the disease.sh ap from https://corona.lmao.ninja to find out the number of worldwide cases, active cases, deaths and recoveries.
